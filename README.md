# Course Registration Form

A simple multi-step course registration form built with HTML, CSS, and vanilla JavaScript.

## ✅ Features

- **Multi-step UI** (Personal Details → Course Details)
- **Client-side validation** using HTML5 form validation
- **Password match validation** (prevents moving forward if passwords don’t match)
- **Auto-fill age** based on the selected date of birth
- **File upload preview** (shows selected file name)
- **Clear all button** resets the form and returns to the first step

## 📁 Project Structure

- `index.html` – Main form markup and JavaScript logic
- `style.css` – Styling and responsive layout

## ▶️ How to Run

1. Open `index.html` in your browser (double-click or right-click → Open with).
2. Fill out the fields and use the **Next** / **Previous** buttons to navigate.
3. Click **Submit Registration** to see the simulated submission alert.

> This project is a front-end demo; form data is not sent to a server.

## 🛠️ Customization Ideas

- Add more detailed country list or course options
- Persist form data in `localStorage` so it isn’t lost on refresh
- Connect to a backend (API) to actually submit registrations
